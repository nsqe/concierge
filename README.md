# concierge
# Get in, loser, we're going vibe coding
This is a project I'm working on with my buddy Claude. Concierge is a personal dining memory app — not a rating app. Users log 
restaurants they've visited, food they ordered, whether they liked it, and notes (e.g., modifications that improved a dish). 
The goal is personal memory and, in a later phase, AI-powered dining recommendations. Users' thoughts and notes will never go to
the restaurant or to advertisers.
